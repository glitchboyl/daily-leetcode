求解一个给定的方程，将x以字符串"x=#value"的形式返回。该方程仅包含'+'，' - '操作，变量 x 和其对应系数。

如果方程没有解，请返回“No solution”。

如果方程有无限解，则返回“Infinite solutions”。

如果方程中只有一个解，要保证返回值 x 是一个整数。

示例 1：

输入: "x+5-3+x=6+x-2"
输出: "x=2"

示例 2:

输入: "x=x"
输出: "Infinite solutions"

示例 3:

输入: "2x=x"
输出: "x=0"

示例 4:

输入: "2x+3x-6x=x+2"
输出: "x=-1"

示例 5:

输入: "x=x+2"
输出: "No solution"
解题思路:
以示例 1 的输入为例，将方程式拆分成 [['x', '+5', '-3', '+x'], ['6', '+x', '-2']]。
将 x 移到等号左边，将常数移到等号右边。为 x 计数，如果 x === 0 的情况下：
1) 常数不为 0，则说明该方程无解。
2) 常数为 0，则说明该方程有无限解。

其他情况，则说明 x 有解。使用常数除以 x 的计数可得出答案。
